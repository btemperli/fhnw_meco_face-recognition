## Installation

* Add Link to the real OpenCV-Library
  * possible link: `/usr/local/Cellar/opencv/2.4.9/share/OpenCV/java/opencv-249.jar`
  * in IntelliJ: `File -> Project Structure -> Libraries -> + -> Java -> go to FolderLink of your opencv`
